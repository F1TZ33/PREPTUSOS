Vehicle Inventory Website

Files:
- index.html
- styles.css
- script.js

How to use:
1. Open index.html in a browser.
2. Add a vehicle.
3. Add locations inside that vehicle.
4. Add items inside each location.
5. Everything saves automatically in your browser using local storage.

Extras:
- Export JSON backup
- Import JSON backup
- Rename and delete vehicles, locations, and items
